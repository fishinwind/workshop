Useful python modules§

There are several modules in the standard library you will use all the time:

  • sys: sys.argv has all the arguments from the command line
  • collections: espcially defaultdict and Counter
  • itertools: tools for efficient aggregation and iteration
  • argparse: command line option parsing

Debugging Python code§

The pdb is the Python Debugger. You can use it to debug programs by dropping you into a shell that allows you to step through the program, line by line.
