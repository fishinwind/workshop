IPython§

Ipython is an (I)nteractive python terminal that lets you type in python expressions and see the results immediately

$ ipython

This command puts you in a shell that accepts python commands, much like the login terminal accepts bash commands.

Python indentation§

Python depends on proper indentation of your code. This does not work:

for i in (1, 2, 3):
print i

Instead you have to indent the print statement to nest it in the for loop:

for i in (1, 2, 3):
    print i

Note

Use spaces instead of tab characters for indentation.

Update your gedit preferences: Edit -> Preferences, check the Insert spaces instead of tabs box.

Python Types§

There are several core types in Python that you will use a lot.

  • str is a collection of characters (words and sentences).
  • int and float are numbers.
  • list is a group of other objects.
  • dict contains key:value mappings.
