Making linked slides and HTML
-----------------------------
Make linked slides with:
    
.. code-block:: bash

    $ make html slides

Content will be in ``_build/html`` and ``_build/slides``.

Making webiste
--------------
Move HTML to github.io site with:

.. code-block:: bash

    $ make website
